MacOS

To begin setup, you will need Administrator privalages. There are 2 methods of installing it, the first is an automated script, the second is manually by hand, entering each line yourself


Step 1: Install Homebrew


Homebrew is a special package manager that is required by a number of other applications, to do this, open terminal, and run the comand below (you can copy and paste it, just hit enter and it will run


/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 2: Adding to path


Now that it has been downloaded and installed, we need to add it to the terminal profile, so that both other applications and ourselves can use it in the future.


Close any terminal windows and open a new one up, and type in `nano .zprofile`

export PATH=/opt/homebrew/bin/:$PATH
