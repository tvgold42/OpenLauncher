Super Mario 64

Whilst one of the more mature projects, this edition requires that the user specify the name of the rom image, check if your rom is the correct version below. This check is ran all on device, so no need to worry about anybody seeing you do anything



The Legend of Zelda: Ocarina of Time

Using the "Ship of Harkinian" project, you can take your existing rom image, and turn the whole game into a native port, this project is still somewhat early in development, but is fully functional with a few selected editions of the game's rom image

The left margin of the page content is set to the same value.



The Elder Scrolls 3: Morrowind

Morrowind has had a complete game engine replacement, and can be installed with either a full game install, the original CD's, or a downloaded copy from GOG.


Jak and Daxter: The Precursor Legacy

This is a community effort to translate the propriatry GOAL programming language used in Jak and Daxter 1 - 3 to run on modern hardware, with the Open Goal compiller. Whilst Jak 2 is currently under way, Jak 1 is ready to play today with the right rom



Lara Croft

This uses the OpenLara engine by XProger to bring the original Lara Croft to modern systems. There are plans for the future titles to be supported, but effort is on going at the moment.

Re-Volt

Whilst not open source, RVGL is a time tested engine replacement for the game Re-Volt. There are no required files or rom images needed to get this setup, but it will download them from Re-Volt IO in order to work.
